# Change Log

## [0.1.0] - 2026-06-30

### Added
- Scaffold new SkillForge skills with directory structure (`skill.yaml`, `skill.py`, `__init__.py`)
- Run installed skills with custom input parameters
- Validate `skill.yaml` manifests for correctness
- Install skills from workspace folder into local registry
- Publish skills to a SkillForge registry server
- List all installed skills in the local registry
- Open SkillForge web dashboard
- Generate OpenAPI specifications from skill manifests
- Syntax highlighting for skillforge-yaml language (injected into YAML files)
- Code snippets for skill creation (skeleton, inputs, outputs, permissions, dependencies, Python run function)
- Configuration settings for registry URL, API key, and Python path
- Context menu integration for explorer and editor
- Keyboard shortcuts for main commands
